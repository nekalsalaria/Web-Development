HTML Interview Questions

1. What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure of web pages.

2. Difference between HTML, CSS and JavaScript?

HTML is used for structure. CSS is used for styling. JavaScript is used for functionality.

3. What is the use of DOCTYPE in HTML?

DOCTYPE tells the browser that the document is written in HTML5.

4. What are semantic tags?

Semantic tags describe the meaning of content.

Examples:

  • header
  • footer
  • section
  • nav

5. Difference between div and span?

div is a block element. span is an inline element.

6. Difference between class and id?

class can be used multiple times. id should be unique.

7. What is the use of forms?

Forms are used to collect user data.

8. Which tag is used to create links?

anchor tag is used.

Example:

<a href="">Click Here</a>

9. Difference between ordered and unordered list?

Ordered list shows numbering. Unordered list shows bullets.

10. What is the use of alt attribute in image tag?

alt attribute shows alternate text if image does not load.
