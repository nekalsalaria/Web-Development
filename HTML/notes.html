HTML Notes

What is HTML?

HTML stands for HyperText Markup Language. It is used to create the structure of websites.

Basic Structure

<!DOCTYPE html>
<html>
<head>
    <title>Page Title</title>
</head>
<body>

</body>
</html>

Heading Tags

h1 to h6 tags are used for headings.

Paragraph Tag

p tag is used to write paragraphs.

Line Break Tag

br tag is used to break line.

Horizontal Line

hr tag creates horizontal line.

Lists

Two Types:

  • Ordered List
  • Unordered List

Anchor Tag

anchor tag is used to create links. 

<a href="https://google.com">Google</a>

Image Tag

img tag is used to show images. 

<img src="image.jpg" alt="image">

Table Tags

  • table
  • tr
  • th
  • td

Form Tags

  • form
  • input
  • textarea
  • button
  • select

Semantic Tags

  • header
  • nav
  • section
  • footer

Difference Between Class and ID

class is reusable. id is unique.
